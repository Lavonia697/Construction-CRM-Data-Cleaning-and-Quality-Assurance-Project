# Construction-CRM-Data-Cleaning-and-Quality-Assurance-Project
## Business Problem
A construction CRM export contained inconsistent project records, duplicate entries, missing values, and multiple date formats that prevented reliable reporting

---

## Cleaning Objectives
- Remove duplicate projects
- Standardize project and client names
- Handle null and blank values
- Convert inconsistent date formats
- Validate cleaned records

  ---

  ## SQL Techniques Used
- CREATE TABLE
- ALTER TABLE
- UPDATE
- DELETE
- CASE
- STR_TO_DATE()
- TRIM()
- UPPER()
- LOWER()
- IS NULL
- ROW_NUMBER()
- CTE

---

## Key Results
✓ Removed duplicate project records

✓ Standardized client names

✓ Converted 3 date formats into a single DATE format

✓ Replaced missing values

✓ Created analysis-ready dataset
